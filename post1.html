    Learning Covenant C2 - Setup Guide & Lessons learned

    45 minute read - Published: 22 Dec, 2023

    > In this guide, I'll show you how to set up Covenant C2 and establish your first listener in under an hour.
    > My lack of experience in .NET and installing dependencies offline turned out to be more time-consuming than necessary. So I aim to spare you this hassle and provide a straightforward, easy-to-follow tutorial to streamline your setup process.
    > As my focus will be to show you what to look out for when installing offline, I'll also walk through setting up on a Kali machine with internet access.
    > As I am still pretty novice to red teaming, I think my perspective be helpful to others who are just starting out as well.

    > Covenant C2 is a command and control (C2) post-exploitation framework. As of mid-September 2019, Covenant is considered one of the latest and advanced frameworks in this area.

    Prerequisites: Compatible with Windows 10 workstations, recent Windows Server OS, or Linux distributions supporting .NET Core. Required applications: GIT and .NET Core 2.2 SDK.

    Installation Process: Covenant uses a modern, cross-platform architecture and does not come with a traditional installer. Installation involves cloning the project git repository and building and running the application.

    Covenant Dashboard Basics

    > So once you verify that Covenant is running, let's talk about the dashboard.
    > Covenant's dashboard is web-based interface, allowing for easy management and deployment of grunts. Grunts are the agents controlled by the Covenant framework, designed to execute tasks and payloads on compromised hosts, reporting back to the Covenant server.
    > Covenant supports a scalable approach, suitable for managing a large number of compromised hosts. Its modular design allows for the addition of new features and capabilities.
    > One key aspect of using C2 frameworks like Covenant is maintaining stealth and avoiding detection. Covenant includes features to evade common security defenses and blend into normal network traffic.
    > As an open-source project, Covenant is regularly updated by the community. Keep in mind, if you are new to using C2 architectures, it's crucial to adhere to legal and ethical guidelines, using it only in authorized environments for security testing and research purposes.
